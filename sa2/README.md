# Sonic Adventure 2: Battle

## Game Info

- **Region:** NTSC-U
- **ID:** GSNE8P
- **MD5:** 9ef5fadf4b8756af820df997468a5a16

## Programs

- [Tales' Dolphin 4.0-4222 + LUA build](https://github.com/THC98/dolphin/releases/latest)
- [Dolphin Memory Engine](https://github.com/aldelaro5/dolphin-memory-engine/releases/latest) (2026.01.24 or later)
- [SA2LevelEditor](https://github.com/TurtleMan64/SA2LevelEditor/releases/latest)

## Usage

- Copy the content of the `scripts` folder into the `Scripts` folder in
your Dolphin directory
- Use `sa2b.dmw` when running Dolphin Memory Engine in order to
inspect important memory addresses more easily

## Scripts

| Script | Description |
| `Sonic_InspectTaskList.lua` | Searches for all objects of a specific type, displaying
their memory address and position on-screen. Currently have to modify the script itself
to configure |
